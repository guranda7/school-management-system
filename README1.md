ძალიან კარგი იდეაა 💛
თემაც ზუსტად ერგება ფინალური პროექტის მოთხოვნებს და რეალურ პრობლემასაც წყვეტს. მოდი, **პირველი ნაბიჯები** ძალიან სწორად და მშვიდად დავალაგოთ, რომ მერე არ აგერიოს არც ლოგიკა, არც ტექნიკური მხარე.

ქვემოთ დავწერ **რა უნდა გააკეთო ჯერ**, *კოდზე გადასვლამდე*.

---

## 🔹 I ეტაპი — იდეის სწორად ფორმულირება (ყველაზე მნიშვნელოვანი)

პირველი ნაბიჯი არ არის კოდის წერა ❌
პირველი ნაბიჯი არის **ზუსტად გაარკვიო რას აკეთებს სისტემა და ვისთვის**.

### 1️⃣ განსაზღვრე მომხმარებლის ტიპები (Roles)

შენ უკვე სწორად ფიქრობ 👌
ამ პროექტში გექნება მინიმუმ **3 როლი**:

1. **ადმინისტრატორი**

   * ყველაფერზე სრული წვდომა
   * მასწავლებლების, მშობლების და ბავშვების მართვა
   * პროექტების/კლასების შექმნა
   * განცხადებების დამატება

2. **მასწავლებელი**

   * თავისი კლასების ნახვა
   * ბავშვებისთვის დავალებების და შეფასებების დამატება
   * კომენტარების დატოვება

3. **მშობელი**

   * მხოლოდ **საკუთარი ბავშვის** მონაცემების ნახვა
   * შეფასებები
   * დავალებები
   * განცხადებები

👉 **ბავშვს (1–4 კლასი)** ცალკე ავტორიზაცია არ სჭირდება – ეს ძალიან რეალისტურიცაა.

## 🔹 II ეტაპი — ფუნქციონალის გადაყვანა „მენეჯმენტის სისტემის ენაზე“

შენი იდეები უკვე ემთხვევა დავალებას, უბრალოდ სწორ ტერმინებში გადავიყვანოთ:

| დავალებაში მოთხოვნილი | შენს პროექტში            |
| --------------------- | ------------------------ |
| Projects              | კლასები / ჯგუფები        |
| Tasks                 | დავალებები               |
| Employees             | მასწავლებლები            |
| Users                 | მშობლები                 |
| Dashboard             | ცენტრის სტატისტიკა       |
| Roles & Permissions   | Admin / Teacher / Parent |

👉 ეს ნიშნავს, რომ **შენ არაფერს არღვევ დავალებაში**, უბრალოდ თემას ცვლი.

## 🔹 III ეტაპი — ძირითადი გვერდების ჩამონათვალი (Wireframe-ივით)

ჯერ არ ვხატავთ დიზაინს, უბრალოდ ჩამოვწეროთ:

### 🔐 ავტორიზაცია

* რეგისტრაცია (მშობელი / მასწავლებელი)
* ლოგინი
* JWT ავტორიზაცია

### 🧑‍💼 Admin Panel

* ყველა მომხმარებლის სია
* კლასების შექმნა (1 კლასი, 2 კლასი…)
* მასწავლებლის მიბმა კლასზე
* განცხადებების დამატება

### 👩‍🏫 Teacher Panel

* ჩემი კლასები
* მოსწავლეების სია
* დავალების დამატება
* შეფასების ჩაწერა

### 👨‍👩‍👧 Parent Panel

* ჩემი შვილი
* შეფასებები
* დავალებები
* განცხადებები
